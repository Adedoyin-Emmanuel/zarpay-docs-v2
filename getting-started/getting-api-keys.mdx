---
title: Getting API Key
description: "This is a guide to get an API key"
---

To create a new API key, you need to sign up. https://zarpay.zarclays.com/sign-up After signing up, you will need to create a store then you can create a new API key.

<Note>
You will need to include the `client-id` header in any request made to our api. The `client-id` is gotten alongside with the **API** key when you create an **API KEY**

</Note>
